# Sobre o projeto

Este projeto Java com Spring Boot e mongoDB é um backend que gerencia posts e comentários de usuários. Foi desenvolvido durante um curso com o objetivo de ensinar os fundamentos do Spring Boot e como integrar o MongoDB como banco de dados.

O backend foi construído utilizando os recursos do framework como a injeção de dependência, as anotações e o padrão REST para criar uma API intuitiva e fácil de usar.

# Tecnologias utilizadas
- Java
- Spring Boot
- Maven
- MongoDB
